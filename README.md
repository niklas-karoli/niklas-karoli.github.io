# Cyber Academy Workshop

Ein interaktiver Workshop für Schüler zum Thema Kryptografie und Binärzahlen.

## 🚀 Deployment auf GitHub Pages

Um die Website unter `https://niklas-karoli.github.io/` (oder deiner Domain) zu sehen, folge diesen Schritten:

1. **GitHub Actions aktivieren**:
   - Gehe in deinem Repository auf den Reiter **Settings**.
   - Klicke links im Menü auf **Pages**.
   - Wähle unter **Build and deployment** bei **Source** die Option **GitHub Actions** aus.

2. **Workflow starten**:
   - Sobald du Änderungen an den `main` Branch pushst, startet automatisch ein Build-Prozess.
   - Du kannst den Status unter dem Reiter **Actions** verfolgen.

3. **Fertig!**:
   - Nach etwa 1-2 Minuten ist die Seite unter deiner URL erreichbar.

## 🛠 Features

- **Welt 1: Kryptografie**: Cäsar- & Vigenère-Verschlüsselung.
- **Welt 2: Binärzahlen**: Bits, Bytes & Binäre Arithmetik (Addition/Subtraktion).
- **Welt 3: Fusion**: XOR-Logik & E2E-Chat Simulation.
- **Welt 4: Python-Workshop**: Interaktiver Code-Editor für Bruteforce-Skripte.
